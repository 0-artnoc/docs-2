# Bitwarden Docs Site

The Bitwarden docs site project is written using Jekyll.

# Build/Run

**Requirements**

- Node.js
- [Ruby](https://www.ruby-lang.org/)
- [Jekyll](https://jekyllrb.com/)

Run the following commands:
- `npm install`
- `npm run build:watch`

You can now access the help center at `http://localhost:4011`.
